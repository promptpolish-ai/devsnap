# DevSnap — Premium UI Components

**10 beautiful, production-ready HTML/CSS components for modern landing pages.**

Dark mode. Responsive. Zero framework lock-in. Effortlessly customizable.

## Components

- Hero Section with particle effects & gradient overlays
- Glassmorphism Navigation Bar
- Feature Grid with hover effects
- Pricing Tables (3 tiers with featured highlight)
- Testimonials Cards
- FAQ Accordion
- Stat Counters
- CTA Sections
- Contact Forms
- Footer with social links

## Usage

1. Open `index.html` in any browser to see the full demo
2. Copy individual component markup from the source
3. Customize colors via CSS variables in `:root`
